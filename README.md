# Online Examination System (CLI)

A command-line based Online Examination System developed using Core Python.

## Features
- Student login using file handling
- Secure access control
- Timed-style exam logic (conceptual)
- MCQ-based questions
- Automatic evaluation
- Score and percentage calculation
- PASS / FAIL result
- Result storage with date & time

## Technologies Used
- Python 3
- File Handling
- Time Module

## Files Used
- `main.py` – main program
- `students.txt` – login credentials
- `results.txt` – exam results

## How to Run
1. Clone the repository
2. Open terminal in project folder
3. Run:
   ```bash
   python main.py
